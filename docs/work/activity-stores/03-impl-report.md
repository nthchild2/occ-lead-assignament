## Deviations from the plan

**None.** All 10 planned changes were implemented as specified. The exact response shapes assumed by the plan (`{ data: Application }` for apply, `{ data: { message: string } }` for cancel/favorite/unfavorite) were confirmed directly against `backend/src/domains/applications/applications.router.ts` and `backend/src/domains/favorites/favorites.router.ts` before writing the local schemas — no mismatch found, so no adjustment was needed.
